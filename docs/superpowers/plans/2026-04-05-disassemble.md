# Disassemble Feature Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a "Disassemble" workflow where the user photographs an appliance's model/serial sticker, OCR extracts the info, and parts pulled from that appliance can be added directly to the catalog with provenance tracking.

**Architecture:** New `appliances` table linked to `parts` via optional FK. Z.ai GLM-OCR model extracts model/serial from sticker photos. Cloudflare R2 stores unit photos with graceful degradation when not configured. Two camera fields on mobile: unit photo (R2, optional) and sticker photo (OCR, required). Frontend gets a new Disassemble tab in the 4-tab bottom nav.

**Tech Stack:** Express + Drizzle (backend), React + Tailwind (frontend), Z.ai GLM-OCR API, Cloudflare R2 (optional), `<input type="file" capture="environment">` for mobile camera.

### Task 3: Backend — OCR endpoint with Z.ai GLM-OCR

**Files:**

The Z.ai GLM-OCR API is at `POST https://api.z.ai/api/paas/v4/layout_parsing` with model `"glm-ocr"`. It accepts a base64-encoded image. The env var is `ZAI_API_KEY`.

- [ ] **Step 1: Create the OCR service**

Create `backend/src/services/ocr.ts`:

```typescript
interface OcrResult {
  modelNumber: string | null;
  serialNumber: string | null;
  brand: string | null;
  rawText: string;
}

/**
 * Send a base64 image to Z.ai GLM-OCR and extract model/serial info.
 */
export async function extractApplianceInfo(imageBase64: string): Promise<OcrResult> {
  const apiKey = process.env.ZAI_API_KEY;
  if (!apiKey) {
    throw new Error("ZAI_API_KEY not configured");
  }

  const response = await fetch("https://api.z.ai/api/paas/v4/layout_parsing", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      Authorization: `Bearer ${apiKey}`,
    },
    body: JSON.stringify({
      model: "glm-ocr",
      content: [
        {
          type: "image",
          image: imageBase64,
        },
      ],
    }),
  });

  if (!response.ok) {
    const text = await response.text();
    throw new Error(`OCR API error: ${response.status} ${text}`);
  }

  const data = await response.json();
  const rawText: string = data.content ?? data.text ?? "";

  // Try regex extraction first
  const parsed = parseModelSerial(rawText);
  if (parsed.modelNumber || parsed.serialNumber) {
    return { ...parsed, rawText };
  }

  // Fallback: use cheap GLM model to extract structured data
  return await fallbackExtract(rawText, apiKey);
}

/** Regex-based extraction of model and serial numbers from OCR text. */
function parseModelSerial(text: string): { modelNumber: string | null; serialNumber: string | null; brand: string | null } {
  let modelNumber: string | null = null;
  let serialNumber: string | null = null;
  let brand: string | null = null;

  // Common label patterns on appliance stickers
  const modelPatterns = [
    /model\s*(?:#|no\.?|number)?[\s:]*([A-Z0-9][\w\-./]{3,30})/i,
    /mod(?:el)?\.?\s*(?:#|no\.?)?[\s:]*([A-Z0-9][\w\-./]{3,30})/i,
    /m\/n[\s:]*([A-Z0-9][\w\-./]{3,30})/i,
  ];

  const serialPatterns = [
    /serial\s*(?:#|no\.?|number)?[\s:]*([A-Z0-9][\w\-./]{3,30})/i,
    /ser(?:ial)?\.?\s*(?:#|no\.?)?[\s:]*([A-Z0-9][\w\-./]{3,30})/i,
    /s\/n[\s:]*([A-Z0-9][\w\-./]{3,30})/i,
  ];

  const brandPatterns = [
    /\b(whirlpool|kenmore|ge|samsung|lg|maytag|frigidaire|bosch|kitchenaid|amana|hotpoint|electrolux|haier|hisense|insignia)\b/i,
  ];

  for (const pattern of modelPatterns) {
    const match = text.match(pattern);
    if (match) { modelNumber = match[1].trim(); break; }
  }

  for (const pattern of serialPatterns) {
    const match = text.match(pattern);
    if (match) { serialNumber = match[1].trim(); break; }
  }

  for (const pattern of brandPatterns) {
    const match = text.match(pattern);
    if (match) { brand = match[1].trim(); break; }
  }

  return { modelNumber, serialNumber, brand };
}

/** Fallback: use a cheap GLM model to extract model/serial from raw OCR text. */
async function fallbackExtract(rawText: string, apiKey: string): Promise<OcrResult> {
  try {
    const response = await fetch("https://api.z.ai/api/paas/v4/chat/completions", {
      method: "POST",
      headers: {
        "Content-Type": "application/json",
        Authorization: `Bearer ${apiKey}`,
      },
      body: JSON.stringify({
        model: "glm-4-flash-250414",
        messages: [
          {
            role: "system",
            content: "You extract appliance information from OCR text. Reply ONLY with valid JSON, no markdown, no explanation.",
          },
          {
            role: "user",
            content: `Extract the model number, serial number, and brand from this appliance sticker OCR text. Reply with JSON: {"modelNumber": "...", "serialNumber": "...", "brand": "..."} — use null for any field you can't find.\n\nOCR text:\n${rawText}`,
          },
        ],
        temperature: 0,
        max_tokens: 200,
      }),
    });

    if (!response.ok) {
      return { modelNumber: null, serialNumber: null, brand: null, rawText };
    }

    const data = await response.json();
    const content: string = data.choices?.[0]?.message?.content ?? "";
    const parsed = JSON.parse(content);

    return {
      modelNumber: parsed.modelNumber || null,
      serialNumber: parsed.serialNumber || null,
      brand: parsed.brand || null,
      rawText,
    };
  } catch {
    return { modelNumber: null, serialNumber: null, brand: null, rawText };
  }
}
```

- [ ] **Step 1: Create the R2 service**

Create `backend/src/services/r2.ts`:

```typescript
import { AwsClient } from "aws4fetch";

let client: AwsClient | null = null;

function getR2Config() {
  const accountId = process.env.R2_ACCOUNT_ID;
  const accessKeyId = process.env.R2_ACCESS_KEY_ID;
  const secretAccessKey = process.env.R2_SECRET_ACCESS_KEY;
  const bucket = process.env.R2_BUCKET_NAME;

  if (!accountId || !accessKeyId || !secretAccessKey || !bucket) {
    return null;
  }

  return { accountId, accessKeyId, secretAccessKey, bucket };
}

function getClient(): AwsClient | null {
  if (client) return client;
  const config = getR2Config();
  if (!config) return null;
  client = new AwsClient({
    accessKeyId: config.accessKeyId,
    secretAccessKey: config.secretAccessKey,
  });
  return client;
}

export function isR2Configured(): boolean {
  return getR2Config() !== null;
}

export async function uploadImage(key: string, data: Buffer, contentType: string): Promise<string> {
  const config = getR2Config();
  if (!config) {
    throw new Error("R2 storage not configured");
  }

  const awsClient = getClient()!;
  const url = `https://${config.accountId}.r2.cloudflarestorage.com/${config.bucket}/${key}`;

  const response = await awsClient.fetch(url, {
    method: "PUT",
    headers: { "Content-Type": contentType },
    body: data,
  });

  if (!response.ok) {
    const text = await response.text();
    throw new Error(`R2 upload failed: ${response.status} ${text}`);
  }

  return key;
}
```

```typescript
import { isR2Configured, uploadImage } from "../services/r2.js";

// POST /api/appliances/upload — upload unit photo to R2
router.post("/upload", async (req, res) => {
  try {
    if (!isR2Configured()) {
      res.status(503).json({
        error: "Image storage not configured",
        message: "R2 storage is not set up yet. Photos cannot be saved at this time.",
      });
      return;
    }

    const { image, contentType } = req.body;
    if (!image || typeof image !== "string") {
      res.status(400).json({ error: "image (base64 string) required" });
      return;
    }

    if (image.length > 10 * 1024 * 1024) {
      res.status(400).json({ error: "Image too large (max 10MB)" });
      return;
    }

    const buffer = Buffer.from(image, "base64");
    const key = `appliances/${Date.now()}-${Math.random().toString(36).slice(2, 8)}.jpg`;
    const type = contentType || "image/jpeg";

    await uploadImage(key, buffer, type);
    res.json({ key });
  } catch (err: any) {
    if (err.message?.includes("not configured")) {
      res.status(503).json({
        error: "Image storage not configured",
        message: "R2 storage is not set up yet. Photos cannot be saved at this time.",
      });
      return;
    }
    console.error("Upload error:", err);
    res.status(500).json({ error: "Upload failed" });
  }
});
```

```typescript
    return new Promise((resolve, reject) => {
      const reader = new FileReader();
      reader.onload = () => {
        const result = reader.result as string;
        // Strip data URL prefix to get pure base64
        const base64 = result.includes(",") ? result.split(",")[1] : result;
        resolve(base64);
      };
      reader.onerror = reject;
      reader.readAsDataURL(file);
    });

      });
```

```typescript
                <div>
                  <label className="block text-xs text-gray-700 dark:text-gray-300 mb-1">Qty</label>
                  <input type="number" min={1} value={partQuantity} onChange={(e) => setPartQuantity(Math.max(1, parseInt(e.target.value) || 1))} className={inputClass} />
                </div>
              </div>
              <div>
                <label className="block text-xs text-gray-700 dark:text-gray-300 mb-1">Description</label>
                <input type="text" value={partDescription} onChange={(e) => setPartDescription(e.target.value)} placeholder="e.g. Door Shelf Bin" maxLength={500} className={inputClass} />
              </div>
              <button type="submit" disabled={addingPart || !partNumber.trim()} className="w-full px-3 py-2 text-xs font-medium rounded-md bg-gray-900 dark:bg-gray-100 text-white dark:text-gray-900 hover:bg-gray-800 dark:hover:bg-gray-200 disabled:opacity-50">
                {addingPart ? "Adding..." : "Add Part"}
              </button>
            </form>
          </div>
        )}

        {/* Parts */}
        {parts.length === 0 ? (
          <p className="text-center py-6 text-xs text-gray-400 dark:text-gray-500">No parts pulled yet</p>
        ) : (
          <div className="space-y-1.5">
            {parts.map((p) => (
              <div key={p.id} className="bg-white dark:bg-gray-900 rounded-lg border border-gray-200 dark:border-gray-700 px-3 py-2.5 flex items-center justify-between">
                <div className="min-w-0">
                  <p className="text-sm font-medium text-gray-900 dark:text-gray-100 font-mono">{p.partNumber}</p>
                  {p.description && <p className="text-xs text-gray-500 dark:text-gray-400 mt-0.5 truncate">{p.description}</p>}
                </div>
                <span className="text-xs font-medium text-gray-500 dark:text-gray-400 tabular-nums shrink-0 ml-3">
                  ×{p.quantity}
                </span>
              </div>
            ))}
          </div>
        )}
      </div>
    </div>
  );
}
```
